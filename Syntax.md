### Legende:
+ mit Zahl zu ersetzen: ``<?>``
+ Deffinition von Bereichen: ``{<?>}``

## Allgemeiner Syntax
+ neue Zeile: ``newline``
+ Trennstrich: ``mline``
+ Bruch: ``<?> over <?>``
+ Hochzahlen (Exponent):
  + einfach: ``^<?>``
  + Bereich: ``^{<?>}``
+ Tiefzahl (Subskript):
  + einfach: ``_<?>``
  + Bereich: ``_{<?>}``
+ Kommandostrich (Notlöstung): ``->``
+ Wurzel:
  + Standard: ``sqrt{<?>}``
  + N-te Wurzel: ❓
